# Career Guidance App

This app helps users explore different career paths, take quizzes to find suitable careers, and interact with a chatbot for personalized advice. It also includes tools like a resume builder and feedback collection.

## Installation

1. Clone this repository
2. Install dependencies: pip install -r requirements.txt
3. Run the app: streamlit run streamlitapp.py
4. Access the app at http://localhost:8501/
